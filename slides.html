대체 도구들의 한계

📝 Notion

대규모 비즈니스 PT에 적당하지 않음
(로딩 속도, 디자인 제약)

🎨 Canva

고품질의 슬라이드 제작 시
수동 작업의 불편함은 여전

🤖 젠스파크, 감마, 마누스 등 AI 에이전트

  • 퀄리티는 우수하나 통제권이 적음: 내 것이 아닌 느낌
  • 복불복의 성격이 강함: 퀄리티의 고저가 뚜렷
  • 아예 초보자가 사용하거나, 고수가 사용해야 퍼포먼스 좋음
  • 무한대로 사용할 수 없는 제약: 사용할 수록 비용 증가

MCP란 무엇인가

🔌 MCP (Model Context Protocol)

AI 모델(Claude)이 외부 애플리케이션과
연결할 수 있게 해주는 도구

📱 USB-C 포트처럼

하나의 표준으로 다양한 도구들을 연결하는 역할
Claude가 파일시스템, 데이터베이스, 웹사이트 등에 접근 가능

📅 2024년 11월 공개

Claude 운영사인 Anthropic에서 오픈소스로 공개
Claude Desktop 앱을 비롯한 AI 도구에서 주로 활용

⚡ 실제 업무 자동화

Claude가 단순 대화를 넘어 파일 관리, 웹사이트 검색,
노션 연동, 디자인 데이터 가져오기 등 수행 가능

슬라이드 제작 - 샘플 제작

1

프로젝트 지식 첨부

생성된 기획안과 디자인 시스템 가이드,
체크리스트를 프로젝트 지식으로 첨부

2

아티팩트로 샘플페이지 제작

HTML, CSS, JS 로 샘플페이지를 제작
(약 4-6 페이지 정도)

3

디자인 가이드 확인

디자인 가이드에 맞는 지 확인 (중요!!)

4

프로젝트 폴더 생성

내 컴퓨터에 프로젝트 폴더 생성:
데스크톱커맨더 MCP가 관리
project plan.md 생성 후 로컬 폴더에 저장

슬라이드 제작 - 전체 슬라이드

✅ 초반 샘플 페이지 검수

원하는 디자인이 나오는 경우

📝 Text-editor MCP 활용

위 아티팩트를 파일 생성 후
위 로컬 폴더에 저장

🔄 부분별 제작

디자인 시스템이 맞춰지면,
부분을 나눠서 제작 진행
작업 완료 시 체크리스트 수정

🔍 검수 과정

부분 완료 후 개별 페이지를
로컬, 모바일(F12 개발자도구)환경에서 검수
